RechargePay Pro 

RechargePay Pro is a modern, high-performance web application designed for managing mobile recharge history and tracking personal or business expenditures. Built with a focus on speed, aesthetics, and user experience, it offers a professional dashboard for financial tracking.

 Features

    Dynamic Dashboard: Track your current budget with a real-time progress bar and interactive "Hero" card.

    Smart Analytics: Automatic calculation of average, minimum, and maximum recharge amounts.

    Dark/Light Mode: Full theme support with smooth CSS transitions for comfortable use in any lighting.

    Advanced Tracking: Add notes to each entry, categorize recharges with presets, and filter your history.

    Custom Notifications: Integrated toast system and a dedicated notification panel for app updates and alerts.

    Team & Comparison: Compare spending patterns across different users or categories with visual bar charts.

 Built With

    Frontend: HTML5, CSS3 (using Syne, DM Sans, and JetBrains Mono fonts).

    Styling: Custom CSS variables for dynamic theming and responsive layouts.

    Logic: Pure JavaScript for state management and real-time UI updates.
